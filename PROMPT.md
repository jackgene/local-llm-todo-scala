We are creating a sample todo list application in Scala. The workspace will consist of three modules, which we will use `sbt` to build and manage. The first module is the backend, and will run the webserver. The second module will be a common module that will contain classes, like ListItem, etc, that are shared between the frontend and backend. The last module will be the frontend. We will use scalajs for our frontend and tyrian as the UI library. This project is being put together as a proof of concept. Please start generating files, code, and tests.
